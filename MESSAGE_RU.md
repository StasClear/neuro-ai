Предлагаю небольшую доработку для SEOline VK-постинга: автоматически добавлять в публикацию ВКонтакте хештеги из стандартного поля `seo_keys`.

Сейчас в материалах InstantCMS ключевые слова уже хранятся в `seo_keys` через запятую. Их можно использовать повторно для автопостинга:

```text
нейросети, искусственный интеллект, AI инструменты
```

В VK-посте это будет превращаться в:

```text
#Нейросети #ИскусственныйИнтеллект #AIИнструменты
```

Логика:

- запятая в `seo_keys` используется как разделитель ключей;
- пробелы внутри одного ключа убираются;
- каждое слово внутри фразы начинается с заглавной буквы;
- к каждому ключу добавляется `#`;
- хештеги добавляются в конец VK-поста отдельной строкой.

Я проверил доработку на рабочем сайте: хештеги корректно формируются и отправляются в VK.

В архиве приложен изменённый файл:

```text
system/controllers/seoline/frontend.php
```

Изменение находится в методе `postToVK()`: перед отправкой через `wall()->post()` к тексту поста добавляется строка хештегов, сформированная из `seo_keys`.

Синтаксис файла проверен на PHP 7.2 и PHP 8.3 через `php -l`.
